# シャドーイング

外側のブロックで定義した識別子を内側のブロックで同じ名前の識別子を定義すると、外側で定義された識別子が「シャドーイング（隠蔽）」される。

```Go
func main() {
  x := 10
  if x > 5 {
    println(x) // 10
    x := 0
    println(x) // 5
  }
  println(x) // 10
}
```

インポートしたパッケージの識別子もシャドーイングできてしまう。

```Go
package main

import "fmt"

func main() {
  x := 10
  fmt.println(x)
  fmt := "おっとー"
  println(fmt) // 「おっとー」が表示される
}
```
